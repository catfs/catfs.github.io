2011 - 2014: Tempus TRUST

2011: GREAT - Graphical Rules Editing and Auditing Tool

    This project goal is to design an environment where business rules can be graphically created and exported to a executable language of an existing open source engine. It provided a software that is currently used and integrated with the other system tools of the company partner of the project. The results of this project are presented in this publication.
